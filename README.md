# Rádio Cultura FM 104.7 App

## Description
The Rádio Cultura FM 104.7 application is designed to provide listeners with seamless access to their favorite radio station anytime, anywhere. Whether you're at home, commuting, or on the go, this app ensures you never miss a beat of your favorite programs.

## Features
- **Live Streaming:** Listen to live broadcasts of Rádio Cultura FM 104.7 in real-time.
- **PWA Support:** The app is built as a Progressive Web Application (PWA), allowing users to install it on their devices directly from the browser.
- **Service Worker Caching:** Efficiently caches audio streams and assets to ensure smooth playback and reduce data usage while offline.
- **Responsive Design:** The interface automatically adjusts to fit various screen sizes, ensuring a great experience on both mobile and desktop devices.
- **iOS/Android Support:** Fully compatible with both iOS and Android platforms, allowing a wide range of users to enjoy the app.
