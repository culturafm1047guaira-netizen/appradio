# App Specification: Rádio Cultura FM 104.7

## Goal
Build a modern, premium web application for Rádio Cultura FM 104.7 that provides a high-quality streaming experience with a stunning user interface.

## Design Principles
1.  **Aesthetics**: Use glassmorphism, vibrant gradients, and smooth transitions.
2.  **Responsiveness**: Mobile-first design that looks great on all devices.
3.  **Interaction**: Intuitive playback controls and visual feedback (animations).

## Core Features
- Live audio streaming.
- Play/Pause toggle with custom animation.
- Volume control.
- Display of the official radio logo.
- Metadata display (if available from stream).

## Technical Requirements
- **Frontend**: React 19, Vite 8, Tailwind CSS 4.
- **Animations**: Framer Motion 12.
- **Audio**: HTML5 Audio API via custom hooks.
- **Structure**: Hybrid 3-layer architecture (Directives, Orchestration, Execution) integrated with Antigravity Kit.
- **Deployment**: Vercel / PWA enabled.
