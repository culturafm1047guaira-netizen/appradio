# UI/UX Standards: Rádio Cultura FM 104.7

## Design System
- **Style**: Glassmorphism (Background blur: 20px, Transparency: 7-12%).
- **Background**: Mesh Gradient (Dark base #0A0A12 with radial highlights in Amber, Crimson, and Magenta).
- **Typography**:
    - **Display**: Syne (Bold/ExtraBold) for headings and branding.
    - **Body**: Nunito for readability and interface elements.

## Color Palette
- **Primary Gradient**: Amber (#F5A623) to Crimson (#E8304A) to Magenta (#D81BCD).
- **Accents**: Royal Blue (#1A3F9E) for secondary interactive elements.
- **Background**: Deep Space (#0A0A12).

## Animation Principles
- **Transitions**: 0.4s ease-out for page transitions.
- **Interactive**: Hover scales (1.05x - 1.1x) with subtle glow effects.
- **Feedback**: Immediate visual response on play/pause toggle.
